## Run Locally

Clone the project

```bash
  git clone https://github.com/rakeshdeka/mern-chat-app.git
```

Go to the project directory

```bash
  cd mern-chat-app
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```
Create .env file and and include
``
PORT=  your port no
MONGOI_URL= 'your mongo url key with username and password'
JWT_SECRET=your_key
``

Start the server

```bash
  npm run start
```
Start the Client

```bash
  //open now terminal
  cd frontend
  npm start
```
